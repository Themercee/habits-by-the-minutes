## Setup

The application use [Tauri](https://tauri.app/) to enable packaging the application, but when running in local, you also can run it with a Web browser.

Using [Bun](https://bun.com/):

```bash
bun run dev
```
or with Tauri:

```bash
bun run tauri dev
```
